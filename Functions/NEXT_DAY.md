# Function Name: NEXT_DAY

### Usage:
Return the date of a specified weekday.
<br><br>

### Compatible Module(s):
ALL
<br><br>

### Signature(s):
```
NEXT_DAY( Date DATE, String WEEKDAY )
```
**Usage:** Return the first date following `DATE` that matches `WEEKDAY`.<br>
**Optional Parameters:**<br>
**Return Type:** `Date`<br>
